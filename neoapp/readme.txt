При запуске проекта в VS code в терминале использовать команду:
pip install -r requirements.txt (посмотрите путь до файла requirements.txt)
.env - файл для подключения к бд, api ключи
requirements.txt - список зависимостей проекта
base_clase.py - 
request.py - модель данных для таблицы Запросы
employee.py - модель данных для таблицы Сотрудники

в бэке работают функции:
аутентификации: при входе считываетя логин и пароль и выдается токин, который содержеит в себе информацию о роли на `сайте`
